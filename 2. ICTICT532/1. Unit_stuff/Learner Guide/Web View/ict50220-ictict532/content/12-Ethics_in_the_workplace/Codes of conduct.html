The purpose of a code of conduct is to provide a clear understanding of the standard of conduct expected when performing duties as an employee of an organisation.

The code of conduct places an obligation on all employees to take responsibility for their own conduct. To achieve this, all employees are expected to familiarise themselves with, and act in accordance with, the code of conduct.

Creating a code of conduct

Creating a code of conduct for an organisation can be a complicated and time-consuming undertaking. Two of the key aspects of this process are outlined below. Select the tabs to learn more.

  • Drafting
  • Adaptation

Creating a code of conduct normally requires a draft, and then possibly a number of revisions until the document meets with satisfaction from everyone involved. Therefore, it would be best to have a collaborative approach to the development of the document, and involve feedback from as many staff members as possible. Also, at every level of the organisation, which should include management, human resources staff, and other appropriate team members.

There are very few businesses that operate exactly the same, therefore you will rarely have a number of organisations that have precisely the same code of conduct documentation. The code of conduct document needs to be adapted to match the different requirements of the organisation and their unique conditions. Over the long term, by understanding the consideration of the different characteristics of the organisation, the code of conduct document will be easier to implement and manage.

Below are some examples of good policies for a code of conduct.

Example

  • What is and is not acceptable behaviour in the workplace.
  • How to treat with equal respect all staff, management, and clients.
  • The importance of good behaviour and the effect it has on the work environment.
  • Examples of expected ethics and values to uphold.
  • How to comply with relevant industry legislation.
  • Clear examples of misconduct, serious misconduct, and the result of such actions.

Important

You should always release a draft of the document so as to obtain feedback from all staff and stakeholders. One final note, always compare the draft code of conduct document to the other organisational policies to make sure they align properly.
