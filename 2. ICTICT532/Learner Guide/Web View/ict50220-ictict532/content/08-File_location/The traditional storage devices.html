Less than 10 years ago the traditional storage location for any files in a business would have been on a hard disk drive somewhere.

On a standalone system this would be an internal Hard Disk Drive (HDD), with backups performed to an external USB device being memory sticks or portable HDD depending on the amount of data.

Select the button below to learn more about different types of storage.

Step forward to today, and with businesses having access to the NBN in most places, cloud storage is now becoming more and more the norm.
