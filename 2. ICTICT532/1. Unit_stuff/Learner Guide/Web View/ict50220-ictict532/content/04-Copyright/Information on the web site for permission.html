When searching for use of resources from a web site it is best to check either the 'Copyright' or 'Terms and Conditions' links normally located on the bottom of the page. With most Australian Government web sites, being state or federal, copying of text and most images are free for use but do contain caveats.

Government and non-government websites usually take different approaches to copyright. Select the headings to learn more.

Government

Normally a government web site will carry the following information.

The Commonwealth owns the copyright in all material produced by this department.

All material presented on this website is provided under a Creative Commons Attribution 4.0 International(this is an external link and opens in a new window) – external site licence, with the exception of:

  • the Commonwealth Coat of Arms
  • this department's logo
  • content supplied by third parties.

The details of the relevant licence conditions are available on the Creative Commons - external site(this is an external link and opens in a new window) website, as is the full legal code for the CC BY 4.0 license. See the links in the panel.

Non-government

Non-government web site

If the web site has a copyright link it will take you to a page where the copyright will be explained. An example is:

Example

Copyright of all original material generated by contributors to the "organisation's name" websites (with the exception of images) on this website is held by "organisation's name".

Copyright on all images on the website remain with the contributor.

You may download, display, print and reproduce written material in unaltered form for your personal, non-commercial use.

If our written material is used on a similar non-commercial website or social media page, then the "organisation's name" websites must be acknowledged as the source

Use of information and/or images for commercial use must be obtained in writing from the "organisation's name" before use, and the website acknowledged as the source.

All other rights reserved.

There will be numerous times where there is no copyright link on the page, however, a terms and conditions link may help with copyright. Most of the time the information will state that all copyright, trademarks, and content are the property of the said organisation.

Normally in the terms and conditions it will state limited use for private no-commercial use, and also to contact them for any clarification.

Important

68868707 / alessandro0770 / stock.adobe.com

If a web site has resources that you would like to use and there is no copyright information anyway, nor is there a way to contact the owner or web site administrator, just do not use any of the resources.

Just because there is no way to verify the copyright does not mean you have permission to use the resources, because you do not.
