Data breaches

A data breach happens when personal information is accessed, disclosed without authorisation, or is lost. Under the notifiable data breaches scheme, you must be told if a data breach is likely to cause you serious harm.

In the reading panel below there are three links to the Office of the Australian Information Commissioner (OAIC) website. The links contain the following content.

  • What is a data breach?
  • What is a notifiable data breach?
  • How you will be told of a data breach.
  • If you think you have not been told about a data breach.
  • How to spot a phishing scam.
  • Civil penalties for data breaches.
