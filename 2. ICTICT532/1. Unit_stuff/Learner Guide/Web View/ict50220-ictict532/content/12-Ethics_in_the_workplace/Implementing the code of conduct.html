It is not enough just to write a code of conduct document for an organisation, it needs to be implemented to have any impact. To implement any type of policy or procedure communication is one of the keys to success. Another key is for management to take a lead role and lead by example.

There are many examples in all industries where management will enforce anti-bullying policies, only to be breaking these guidelines themselves. Some organisations will develop and implement a range of policies and procedures so as to gain government contracts, with no intentions of ever abiding by them.

Management needs to uphold the same values and ethics too. There are many ways to communicate the code to staff, however, be mindful of language barriers and the availability of information as obstacles to effective implementation. Selecting the right channels to get the message across is important.
