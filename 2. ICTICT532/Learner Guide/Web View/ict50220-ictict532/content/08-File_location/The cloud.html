A simple cloud solution for a micro or small business that has less than 1TB of data could be the use of services such as OneDrive. The files would be located on a drive at the business premises and simply synchronised with the cloud service. However, for a more secure option it would be preferrable to look at cloud services offered by the numerous vendors, currently in the marketplace.

Cloud computing helps organisations to ease the burden of managing IT infrastructure and saves money on capital expenditure spent on infrastructure. Today’s organisations can implement their complete IT network and run their applications on the cloud. There are excellent features available in cloud services, such as high availability and scalability, which companies can do in a few clicks, rather than spending hours configuring and managing them.

In this section we will be only looking at cloud as a storage service, although the basics will still be covered.

Cloud computing models

— 1415429273 / Luna2631 / shutterstock.com, Modified by TAFE Queensland

Software as a Service (SaaS)

Software as a service enables consumers to use software programs in the cloud without installing them in local computers. SaaS already includes IaaS and PaaS, but consumers only use and pay for SaaS built on IaaS and PaaS.

Best example for SaaS is Microsoft Office 365. Consumers can use complete Microsoft Office package online by subscribing to MS Office 365 and pay for monthly subscription.

Platform as a Service (PaaS)

Platform as a service includes the preconfigured environment of IaaS. Cloud consumers only use the service as a platform to run their development tools, use operating systems, and for business analytics.

In this model, the burden of infrastructure management is taken away from the consumers and consumers only pay for the platform services they use.

Infrastructure as a Service (IaaS)

Hardware based IT resources in a network can be categorised as infrastructure. This may include computers (servers), storage, and networking resources such as routers and switches (a device that connects routers together). Cloud can host these resources in a virtual environment where consumers can implement their complete network in the cloud which is called Infrastructure as a Service.

IaaS offered in the cloud is not a pre-configured service; consumers have full control over how they want the network and infrastructure deployed, therefore the responsibility of managing the cloud is placed directly upon cloud consumers.
